## ⚙️ Example Profile Configuration

Here’s a sample YAML configuration for a user profile. This approach allows ultra-granular control over how your codes are managed:

    profile:
      username: "GamerZenith"
      encrypt_vault: true
      preferred_language: "en"
      redemption_policies:
        max_uses_per_month: 5
        auto_transfer_unused: true
        notify_expiry_days: 14
      backup_emails: 
        - "youremail@provider.com"
      linked_accounts:
        discord: "Zenith#1234"
        xbox: "GamerZenithXBL"

Feel free to create multiple profiles for friends/family or community management!
